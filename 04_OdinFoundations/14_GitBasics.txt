How do you create a new repository on GitHub?

- + button at the top of the webpage

How do you copy a repository onto your local machine from GitHub?

- git clone git@github.com:EXAMPLEuser/EXAMPLErepo.git

What is the default name of your remote connection?

- origin

Explain what origin is in git push origin main.

- It is the initial location of commits to push over and add to main's commits

Explain what main is in git push origin main.

- It is the receiving remote repo that takes origin's commit history and adds it to its own

Explain the two-stage system that Git uses to save files.

- You move the changed file into the staging area (making it staged) and wrap up the changes with a commit message (making a commit)

How do you check the status of your current repository?

- git status

How do you add files to the staging area in git?

- git add

How do you commit the files in the staging area and add a descriptive message?

- git commit -m

How do you push your changes to your repository on GitHub?

- git push origin main

How do you look at the history of your previous commits?

- git log (or the GitHub page itself)
