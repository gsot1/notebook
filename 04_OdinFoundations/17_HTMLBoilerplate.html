What is the purpose of the doctype declaration?

It specifies what version the HTML file should adhere to

What is the HTML element?

It denotes the section that contains the actual HTML content in the file

What is the purpose of the head element?

It stores HTML tags that don't result in items being displayed on-screen

What is the purpose of the body element?

It stores HTML tags that DO get displayed on screen, it's the main event of the file
