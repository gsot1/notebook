What kind of program is Git?

- Git is a version control system that stores checksummed snapshots of changes to a codebase

What are the differences between Git and a text editor in terms of what they save and their record keeping?

- A text editor overwrites the single file it edits while Git stores a backlog of every version of that file

Does Git work at a local or remote level?

- Both (copying from a remote Git repo stores a local copy on your computer)

Does GitHub work at a local or remote level?

- Only remote (GitHub is a service that stores repos)

Why is Git useful for an individual developer?

- Helps you track all of your changes and revert to old versions if you need to

Why are Git and GitHub useful for a team of developers?

- It helps keep track of everyone's contributions and merging them together without overwriting anything important
